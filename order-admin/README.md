# 订单后台界面

## 界面功能

- [x] 员工管理与权限控制
- [x] 订单管理
- [x] 产品管理
- [x] 合同管理
- [x] 客户管理，包含客户标签及付款渠道
- [x] 其它功能：数据库导入与导出


``` bash
# 安装项目依赖
npm install

# 启动项目于localhost:8080
npm run dev

# 项目打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
