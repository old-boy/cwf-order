# 订单接口API

## 一、这是个什么的项目？

基于 Node.js express 实战开发的一套完整的订单项目，适合多端的 RESTful API，同时配套完整的后台管理系统，基于 Vue.js 前后端分离的前端界面。
为公司日常订单统计表，有员工权限，客户标签，汇款方式，订单统计等日常登记表。
